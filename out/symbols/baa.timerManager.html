<static> {baa.timer} baa.timerManager.newTimer(obj, time, mode, func, cond)
Create a new timer belonging to this timer manager.
Parameters:
{object} obj Optional
The object you want to connect to the condition and function of this timer. If the first argument is a number, this will be taken as the argument time, and shift all other arguments, and the manager's object will be used instead.
{number} time
The length of the timer in seconds.
{string} mode Optional, Default: "normal"
If the timer should loop, die after one loop, or neither. Options: "loop", "once", "normal".
{function|string} func Optional
The function you want to call each time the timer is finished. This can either be the name of the function as a string, or a function on its own.
{object|function} cond Optional
A condition that must be true for the timer to play. This can either be a function returning a boolean, or an object with properties that have to be a certain value.
Returns:
{baa.timer} The new timer.
