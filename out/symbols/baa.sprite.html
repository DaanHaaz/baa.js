Method Detail
<static> baa.sprite.addAnimation(name, frames, speed, mode, semi)
Adds an animation
Parameters:
{string} name
The name of the animation
{number[]} frames
The frames the animation. Use numbers between 1 and the number of frames.
{number} speed Optional, Default: 12
The speed the animations plays with
{string} mode Optional, Default: "loop"
The loop mode. "loop" for looping, "once" for the animation playing only once, and "pingpong" for the animation to go back and forth.
{number} semi Optional, Default: 0
When ended, and looping, where the frame should start from instead. Useful for animations with a starting animation you only want to play once.

<static> baa.sprite.setImage(url, width, height, smooth)
Sets the given image. Makes frames based on the given width and height. Automatically centers the origin based on the width and height. 
//The sprite's width and height become that of the image.
this.setImage("player");

//the image is cut down to smaller images, frames, that can be used for animations.
this.setImage("player",16,16);
Parameters:
{string} url
The url of the string
{number} width Optional
The width of the frames
{number} height Optional
The height of the frames
{boolean} smooth Optional
If the image should use smoothing when drawing
