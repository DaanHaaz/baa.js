Class Detail
baa.timer(time, mode, obj, cond, func, manager)
A tool to make easy timers. You can make them loop, give them condiitons for playing, and make the call a function when they're done.
Parameters:
{number} time
The length of the timer in seconds.
{string} mode Optional, Default: "normal"
If the timer should loop, die after one loop, or neither. Options: "loop", "once", "normal".
{object} obj Optional
The object you want to connect to the condition and function of this timer.
{object|function} cond Optional
A condition that must be true for the timer to play. This can either be a function returning a boolean, or an object with properties that have to be a certain value.
{function|string} func Optional
The function you want to call each time the timer is finished.
{object} manager Optional
The manager that owns this timer. Should only be given by the manager itself. This can either be the name of the function as a string, or a function on its own.
See:
baa.timerManager
