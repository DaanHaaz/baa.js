Class Detail
baa.button(x, y, obj, func, shape, onRelease)
A button that can call functions when you click on it. You can make the shape of the button either squared or round. By default, buttons won't activate until you release them.
Parameters:
{number} x
The horizontal position of the button.
{number} y
The vertical position of the button.
{object} obj Optional
The object that is used for calling the function.
{function|string} func Optional
The function you want to call when the button is pressed. This can either be the name of the function as string, or a function on its own.
{string} shape Optional
The shape of the button. The default shape can be set by changing baa.button.defaultShape.
{boolean} onRelease Optional, Default: true
If the button should be activated on release. If this is false, it will be activated on press.
