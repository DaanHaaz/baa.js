<static> baa.graphics.draw(img, quad, x, y, r, sx, sy, ox, oy, kx, ky)
Draws an image. If the second argument is an object, a quad, it will draw only part of the image. Else, it will assume the second argument is x.
Parameters:
{baa.graphics._image|string} img
The image you want draw, or a string of the name of the image.
{object} quad Optional
An object with an x, y, width and height. 0, 0, 32, 32 would be the top left part of the image, and a 32x32 rectangle. Using a number here will take this argument as x, and shift all the arguments.
{number} x Optional, Default: 0
The horizontal position
{number} y Optional, Default: x
The vertical position
{number} r Optional, Default: 0
The rotation in radians
{number} sx Optional, Default: 0
The horizontal scaling
{number} sy Optional, Default: sx
The vertical scaling
{number} ox Optional, Default: 0
The horizontal origin. The origin point is from where the text rotates and scales.
{number} oy Optional, Default: ox
The vertical origin. The origin point is from where the text rotates and scales.
{number} kx Optional, Default: 0
The horizontal shearing factor
{number} ky Optional, Default: kx
The vertical shearing factor
Throws:
{Error}
If img is a non-existing image.
